# Sleep Stages Classification using EOG Signals

## Overview
This project focuses on automatic sleep stage classification using Electrooculography (EOG) signals and deep learning techniques.
The model processes EOG signals to classify different sleep stages by extracting temporal and spatial features from the data.

## Project Workflow
1. Signal Acquisition  
2. Preprocessing and Normalization  
3. Epoch Segmentation  
4. Feature Extraction using CNN layers  
5. Temporal Learning using LSTM  
6. Sleep Stage Classification using Dense layers  
7. Performance Evaluation

## Model Architecture
The implemented deep learning model consists of
- Convolutional Neural Network (CNN) layers for feature extraction
- Max Pooling layers for dimensionality reduction
- LSTM layer for temporal dependency learning
- Dense layers for final classification
